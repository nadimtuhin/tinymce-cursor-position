# Contributing

Thanks for taking the time to contribute!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/tinymce-cursor-position.git`
3. Create a feature branch: `git checkout -b my-feature`
4. Make your changes
5. Commit: `git commit -m "feat: describe your change"`
6. Push: `git push origin my-feature`
7. Open a Pull Request

## Guidelines

- Keep changes focused and minimal
- Update README.md if behaviour changes
- Open an issue first for large changes

## Reporting Bugs

Use the [issue tracker](https://github.com/nadimtuhin/tinymce-cursor-position/issues) and include:
- TinyMCE version
- Browser and OS
- Steps to reproduce
- Expected vs actual behaviour
